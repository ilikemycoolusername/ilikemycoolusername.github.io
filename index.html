My Homepage

This webpage showcases a variety of three-dimensional models which I created using an open-source software called "Blender." The majority of these models were created during July and August of 2022. I followed several different YouTube tutorials to acquire knowledge on how to create the models. All of the models are "low poly" (referring to low polygon), which means that they have limited detail. While I intended to implement these models into a game that I wanted to create at the time, the game currently remains unfinished.
