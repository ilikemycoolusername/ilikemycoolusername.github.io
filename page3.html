Cacti

About:

In order to create a cactus, start by removing the default Cube mesh. Now add a Cylinder mesh. Decrease the number of vertices to a small odd number. Now, change into edit mode by pressing tab. Scroll around your cube and press option to snap to an orthographic view. Press option again before clicking on the top of the cylinder. This will select the ring of vertices around the top of the cylinder. Extrude several times by pressing E. You can change the size of sections of the cactus by using S to scale after you have extruded a new face. Now select certain sections of the cactus by again pressing option then clicking. Use G to move each section so that the cactus is not perfectly straight. You may want to create a curved shape. To add branches to the cactus, repeat the above process with a new mesh. Alternatively you could duplicate the current mesh by pressing shift + D, but this is unlikely to yield the best resulting model. Finally, to add spikes, insert a Cone mesh. Decrease the number of vertices to a small odd number. Scale the size of the Cone to your liking by using S. If you press X after pressing S, then you can scale exclusively along the x-axis (the same is true with y and z). Put several of the cones around your cactus for a spiky look.
